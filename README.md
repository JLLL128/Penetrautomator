# Penetrautomator
An automated penetration testing tool

## 目前实现功能

- CMS查询（调用whatcms的api）
- 路径爆破（调用dirsearch）
- 端口扫描（调用nmap的命令）
- 子域名爆破（调用subDomainBrute）
- ip扫描（模仿fscan的扫描方法）
- **域名反查公司名（调用whois的api）**
- 日志生成

## 用法

Penetrautomator.py是命令行模式

```
#在终端里面输入
python Penetrautomator.py -u target_url --all //完整扫描
python Penetrautomator.py -u target_url --cms //只查询cms
#-u是必加参数，后面的参数不带默认完整扫描
```
